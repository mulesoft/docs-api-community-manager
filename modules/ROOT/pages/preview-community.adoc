To show a live preview of the community's site, including APIs in *Preview Mode*, open the ACM administrator interface and select *Community Builder*.

image::builder.png[Site Builder]

Select *Preview*.

To return to the site builder, select *Back to Builder*.

Use *Community Builder* to *Publish* the community.

Open the ACM administrator interface, select *Community Workspace*, and make the community *Active*.

To view the live site when the community is published and active, open the ACM administrator interface and select *Open Community*.
