Use these sources to stay informed about new releases and downtime for Anypoint API Community Manager.

== Release Announcements

Upgrades are applied automatically to Anypoint API Community Manager versions 2.0 and later.

Details about new releases are announced at http://trust.mulesoft.com in the "Scheduled Maintenance" section and in xref:release-notes::api-community-manager/api-community-manager-release-notes.adoc[the API Community Manager release notes].

For further information, contact your MuleSoft customer success representative.

== Downtime

MuleSoft and Salesforce report and monitor platform downtime at http://trust.mulesoft.com and http://trust.salesforce.com.
